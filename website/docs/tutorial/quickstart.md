# Quickstart

Start with the recording generated by bootstrap. It is intentionally small: one
Markdown file, one shell script, and one expected line of output.

```bash
studio action=bootstrap
studio recording=quickstart action=build
```

The `quickstart` recording is not checked into this repository as source. It is
created by `action=bootstrap` in the current recording workspace.
